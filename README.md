# Sérendipité

## How?

```sh
./autogen.sh
./configure --enable-debug --enable-colors
make -f Makefile_spotify
make
```

## What?
A last.fm / Spotify mashup software written in C, that intends to create dynamic
Spotify playlists based on you last.fm recommendations.
